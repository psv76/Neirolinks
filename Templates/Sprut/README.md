# Шаблоны Sprut.hub

Папка содержит шаблоны устройств Sprut.hub для интеграции с виртуальными устройствами Wiren Board.

Шаблоны используются для отображения устройств, которые создаются в WB через `wb-rules` и передаются в Sprut.hub по MQTT.

## Разделы

```text
Templates/Sprut/
├─ Heating/
├─ Ventilation/
└─ Gates/
```

## Назначение разделов

- `Heating/` — отопление, термостаты и климатические зоны.
- `Ventilation/` — будущие шаблоны вентиляции.
- `Gates/` — будущие шаблоны ворот, калиток и приводов.

## Правило размещения

Подробное описание конкретного шаблона должно лежать внутри папки соответствующей инженерной системы.

Например, описание шаблонов термостатов находится здесь:

Templates/Sprut/Heating/README.md

## Связь с Wiren Board

Sprut.hub используется как интерфейс:

```text 
Sprut.hub = отображение, уставки, режимы, пользовательское управление
```

Логика инженерной системы остаётся в Wiren Board:

```text 
Wiren Board = логика, защита, исполнительные команды, журнал
```

Шаблон Sprut.hub не должен подменять логику WB-скрипта.
