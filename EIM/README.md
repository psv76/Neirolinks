## Метод инженерной интеграции


**EIM (Engineering Integration Method)** — метод организации инженерной интеграции систем автоматизации, основанный на единой модели объекта, формируемой из проектных данных и используемой для генерации документации, конфигурации оборудования и программной логики.

Система организации работы интегратора, объединяющая:

- инженерную документацию
- модели системы
- автоматизацию конфигурации
- программные скрипты
- накопление решений и опыта объектов
