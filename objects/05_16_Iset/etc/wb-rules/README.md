# 05_16_Iset / wb-rules

Эта папка — общий навигатор по сценариям `wb-rules` объекта `05_16_Iset`.

Структура папки соответствует расположению файлов на контроллере Wiren Board в `/etc/wb-rules`: эксплуатационные JS-скрипты лежат рядом и не разделяются по подпапкам `Heating` или `Ventilation`.

## Основные JS-файлы

- `Boiler_room.js` — мониторинг и диагностика котельной.
- `DHW_priority_manager.js` — приоритет ГВС и пауза отопительных контуров.
- `Floor_mixing_control.js` — управление узлом подмеса тёплого пола.
- `Heating_pumps_manager.js` — управление насосами отопления жилой части.
- `Basement_radiator_thermostat.js` — временный термостат радиаторов цоколя.
- `Vilpe_control.js` — управление вытяжкой Vilpe и режимом сушки сауны.
- `Light_4.js` — сценарий освещения.
- `Master-Light.js` — мастер-сценарий освещения.

## Подробные описания

Подробные описания отдельных групп сценариев лежат в отдельных `.md` файлах рядом со скриптами:

- [Heating.md](Heating.md) — отопление, ГВС, насосы и подмес.
- [Vilpe_control.md](Vilpe_control.md) — вытяжка Vilpe, интеграция с Breezart и сушка сауны.
